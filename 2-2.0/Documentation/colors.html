SCSS Variables

Following colors are available in SCSS variables.

$white: #fff;
$black: #000;
$teal: #53ffc1;
$purple: #ac66f5;
$red: #E53935;
$amber: #ffc021;
$blue: #27a4fb;
$green: #2cc56f;
$cyan: #00f3e7;
$yellow: #ffe21f;
$lime: #8BC34A;
$orange: #FF5722;
$pink: #E91E63;
$indigo: #3F51B5;
$brown: #795548;
$blue-grey: #607D8B;

You can utilize these variables when needed by setting them as properties in CSS classes/IDs.

.highlight {
    color: $red;
    background-color: $green;
    border: 1px solid $blue;
}
