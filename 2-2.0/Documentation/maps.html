JQVMaps

JQVMap is a plugin that renders Interactive, Clickable Vector Maps. It uses resizable Scalable Vector Graphics (SVG) for modern browsers.

Page Demo Plugin Website

Available Maps

Select a vector map from the dropdown below to reveal it's code. Make sure to include the map script file which listed below the map.

World
World Asia Africa Europe Australia North America South America USA Canada Russia Germany France
  • HTML
  • Javascript
<div class="jqvmap"></div>
if($('.jqvmap')[0]) {
    $('.jqvmap').vectorMap({
        map: [map-name], // Pick a map from below table
        backgroundColor: 'rgba(0,0,0,0)',
        color: 'rgba(255,255,255,0.5)',
        borderColor: 'rgba(255,255,255,0.5)',
        hoverColor: 'rgba(255,255,255,0.75)',
        selectedColor: '#fff',
        enableZoom: true
    });
}
