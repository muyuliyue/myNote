Containers

Demo Default Demo Small Container Demo Extra Small Container

In addition to the default end-to-end container layout, there are 2 additional smaller containers available in order to fit any contents which needs a smaller layout with compact boundary.

Please note that small container is only visible in larger desktop monitors.

  • Default
  • Small
  • Extra Small

This is the default end-to-end container layout and this does not require any code changes.

<div class="main">
    <aside class="sidebar navigation">
        <div class="scrollbar">
            <!-- Nvigation content -->
        </div>
    </aside>

    <section class="content">
        <header class="content__title">
            <h1>Main heading</h1>
        </header>

        <!-- Page contents -->

        <footer class="footer">
            <!-- Footer contents -->
        </footer>
    </section>
</div>

For small container, add the following child wrapper inside the element .content

<div class="content__inner">
    <!-- Page contents -->
</div>
<div class="main">
    <aside class="sidebar navigation">
        <div class="scrollbar">
            <!-- Nvigation content -->
        </div>
    </aside>

    <section class="content">
        <div class="content__inner">
            <header class="content__title">
                <h1>Main heading</h1>
            </header>

            <!-- Page contents -->

            <footer class="footer">
                <!-- Footer contents -->
            </footer>
        </div>
    </section>
</div>

For extra small container, add modifier class .content__inner-sm on element .content__inner which was previously added for small container.

<div class="content__inner content__inner--sm">
    <!-- Page contents -->
</div>
<div class="main">
    <aside class="sidebar navigation">
        <div class="scrollbar">
            <!-- Nvigation content -->
        </div>
    </aside>

    <section class="content">
        <div class="content__inner content__inner--sm">
            <header class="content__title">
                <h1>Main heading</h1>
            </header>

            <!-- Page contents -->

            <footer class="footer">
                <!-- Footer contents -->
            </footer>
        </div>
    </section>
</div>
